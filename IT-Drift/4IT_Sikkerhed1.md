# IT-Sikkerhed 1

## CIA-triad

* Confidence - Only intended people can see the data that is protected
* Integrity - Data isn't altered in any forms. Data is Identical.
* Availabilty - Stable, Persistent systems

## The Unintended Threats

* The outcome of intended vs uninteded threats are the same.
* Power failure, Fire hazards, weather (earthquake, floods), hardware craches, human errors
* Therefore we need precautions

1. UPS (Uninterruptible power-supply)
2. Physical metal doors
3. Inactive gas
4. Airconditioning for temp/humidity (25 degrees celcius, 55% humidity)
5. Early warnings => When the UPS starts, when temps drop or rise, etc.
6. Remote backups
7. Split operations (2 sites)

* All precautions wont be necessary each time, example: an UPS is not enough for Rigshospitalet.
